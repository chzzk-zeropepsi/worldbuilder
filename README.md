# Worldbuilder 🌐 — AI 소설 설계 워크벤치

소설 **세계관·인물·플롯**을 처음부터 함께 설계하는 도구입니다.
AI(로컬 LLM)가 아이디어를 확장·제안하고, **당신이 직접 고치고 키워나갑니다.**
AI는 작가를 대체하지 않습니다 — 브레인스토밍 파트너입니다.

- 🎬 **기획 스튜디오** — 장르 → 로그라인 → 테마 → 클리셰 → 인물 → 세계 → 플롯, 단계별로 제안받기
- 🌐 **세계관 바이블** — 설정을 카드로 정리, 설정 충돌 자동 검사
- 📅 **연표** — 사건을 시간순으로
- 🕸️ **관계도** — 인물·세력 관계를 그래프로 (AI 자동 분석)
- 📚 **여러 작품**을 따로 관리

> 모든 것은 **내 컴퓨터 안에서만** 돌아갑니다. 인터넷에 글이 올라가지 않고, 사용료도 없습니다.

---

## 🖥️ 준비물 (컴퓨터 사양)

- **Windows 10 / 11**
- **그래픽카드 메모리(VRAM) 8GB 이하**로도 동작 (AI 모델이 가볍습니다)
- 저장공간 약 **6GB** (AI 모델 다운로드용)

---

## 📦 설치 (처음 한 번만, 약 10~20분)

컴퓨터를 잘 몰라도 따라 할 수 있게 **순서대로** 적었습니다. 하나씩 그대로 하세요.

### 1단계. 파이썬(Python) 설치
프로그램을 돌리는 엔진입니다.

1. https://www.python.org/downloads/ 접속 → 노란 **Download Python** 버튼 클릭
2. 내려받은 파일 실행
3. ⚠️ **중요**: 첫 화면 맨 아래 **“Add python.exe to PATH”** 체크박스를 **꼭 켜고** → **Install Now** 클릭
4. 끝나면 닫기

### 2단계. 오라마(Ollama) 설치 — AI 두뇌
AI를 내 컴퓨터에서 돌려주는 프로그램입니다.

1. https://ollama.com/download 접속 → **Download for Windows** 클릭
2. 내려받은 파일 실행해서 설치 (계속 다음 누르면 됨)
3. 설치되면 작업표시줄 오른쪽 아래에 라마(🦙) 아이콘이 생깁니다. 항상 켜져 있어야 합니다.

### 3단계. AI 모델 내려받기
한국어 소설 창작에 강한 모델을 받습니다.

1. 키보드 **⊞ Windows 키 + R** → `cmd` 입력 → 엔터 (검은 창이 뜹니다)
2. 아래를 그대로 복사해 붙여넣고 엔터:
   ```
   ollama pull exaone3.5:7.8b
   ```
3. 4.8GB라 인터넷 속도에 따라 몇 분 걸립니다. `success`가 나오면 완료. 창은 닫아도 됩니다.

### 4단계. Worldbuilder 코드 내려받기
1. 이 페이지 위쪽 초록색 **`< > Code`** 버튼 → **Download ZIP**
2. 내려받은 ZIP을 **압축 풀기** (예: 바탕화면에)

---

## ▶️ 실행 방법 (매번 이렇게)

1. 압축 푼 **`worldbuilder` 폴더**를 엽니다.
2. 그 안의 **`run.bat`** 파일을 **더블클릭**합니다.
   - 처음 실행하면 필요한 부품을 자동으로 설치하느라 1~2분 걸립니다. 검은 창이 떠 있어도 정상이니 닫지 마세요.
   - `Application startup complete` 같은 글자가 보이면 준비 완료입니다.
3. 웹브라우저(크롬/엣지) 주소창에 아래를 입력하고 엔터:
   ```
   http://localhost:8770
   ```
4. 화면 오른쪽 위에 **● 연결됨** 이라고 초록색으로 뜨면 AI까지 정상입니다.

> **끝낼 때**: 검은 창(run.bat)을 닫으면 서버가 꺼집니다. 작업 내용은 자동 저장되어 다음에 그대로 이어집니다.

## 🧭 사용법 한눈에

| 탭 | 설명 |
|----|------|
| 🎬 기획 스튜디오 | 왼쪽 단계를 따라가며 **💡 제안받기**. 마음에 들면 **채택/추가**, 아니면 **거절**, 글자는 직접 **고쳐서** 쓸 수 있어요 |
| 🌐 세계관 바이블 | 설정을 카드로 저장. 저장하면 AI가 **설정 충돌(모순)**을 자동으로 잡아줍니다 |
| 📅 연표 | 사건을 시간순으로 정리 |
| 🕸️ 관계도 | 인물/세력 관계를 그래프로. **🤖 AI 관계 자동 분석**으로 관계를 제안받고 골라 담기 |

- 오른쪽 위 **작품 드롭다운**으로 여러 소설을 따로 만들고 전환할 수 있습니다 (`＋` 새 작품, `✎` 이름변경, `🗑` 삭제).

---

## ❓ 잘 안 될 때

| 증상 | 해결 |
|------|------|
| 오른쪽 위가 **● Ollama 꺼짐** | 라마(🦙) 아이콘이 켜져 있는지 확인. 없으면 Ollama를 다시 실행 |
| **● 모델 없음** | 3단계의 `ollama pull exaone3.5:7.8b`를 다시 실행 |
| run.bat이 **깜빡하고 바로 꺼짐** | 파이썬 설치 때 “Add to PATH”를 안 켰을 수 있어요. 1단계를 다시 설치 |
| 브라우저에 **연결할 수 없음** | run.bat 검은 창이 아직 켜져 있는지 확인. 잠시 기다렸다 새로고침 |
| 화면이 **그대로** | 브라우저에서 **Ctrl + F5** (새로고침) |

## 🔧 개발자용 (직접 명령어로 실행)

```bash
# 1) 모델 준비
ollama pull exaone3.5:7.8b

# 2) 의존성 설치
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

# 3) 서버 실행
cd server
uvicorn main:app --port 8770 --reload
```

**구성**: FastAPI(`:8770`) + SQLite(`worldbuilder.db`) + Ollama(`exaone3.5:7.8b`) + 정적 브라우저 UI.
데이터는 `worldbuilder.db` 파일에 저장됩니다(작품·설정·관계 전부). 지우면 초기화됩니다.
